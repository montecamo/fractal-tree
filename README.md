## Features

- **lots of settings:** 6 different settings for amazing patterns
- **smooth animations:** for visual delight
- **convenience:** supports dragging and zooming

## Example

```js
var FractalTree = require('FractalTree');

const FractalTree = new Tree({
  color: '#ffd5d5',
  width: 200,
  height: 200,
  topOffset: 200 - 40,

  depth: 10,
  angle: 40,
  length: 100,
  lengthRatio: 0.1,
});
```
